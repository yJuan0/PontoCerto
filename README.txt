O projeto utiliza o banco de dados MySQL para armazenar o histórico de buscas realizadas pelos usuários.
A tabela historico_buscas contém as informações da linha consultada, o sentido da viagem, o tempo previsto e a data/hora da consulta.

Script de criação: disponível no arquivo database/banco_ponto_certo.sql.

Essa estrutura permite validar o requisito de integração entre backend e repositório de dados, garantindo persistência mínima para a prova de conceito.
